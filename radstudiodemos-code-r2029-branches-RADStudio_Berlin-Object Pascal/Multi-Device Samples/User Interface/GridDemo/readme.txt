Grid Demo

Native rendering support for TGrid on iOS was added in Berlin Subscription Update 1. To enable native grid rendering for TGrid and TStringGrid on iOS, you will need to select ControlType=Platform and manually add the FMX.Grid.iOS.pas file to the project. 

For more information on this demo project, please visit http://docwiki.embarcadero.com/CodeExamples/Berlin/en/FMX.GridDemo_Sample
