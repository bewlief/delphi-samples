Open Tools API

Description This sample package demonstrates how to create new wizards using the Experts components. These components are used internally by Embarcadero to build WebBroker and DataSnap wizards.

You will need to install the packages dclExpertsUI160.bpl and dclExpertsCreators160.bpl in order to open the designers in this project.

To use the wizards, build the SampleExpertsPackage1 and install it into RAD Studio. Use File | New | Other... to invoke the wizards.
